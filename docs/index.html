0.1 项目简介

0.4 运行方式

环境要求

Python 3.11+，依赖包见 requirements.txt。首次运行需安装 BaoStock 和 AKShare。

# ① 安装依赖
pip install -r requirements.txt

# ② 依次运行三个 Notebook
jupyter nbconvert --to notebook --execute --inplace 01_download.ipynb
jupyter nbconvert --to notebook --execute --inplace 02_clean.ipynb
jupyter nbconvert --to notebook --execute --inplace 03_analysis.ipynb

# ③ 生成最终报告
python -m jupyter nbconvert --to html 03_analysis.ipynb \
    --output report.html --no-input --embed-images

# ④ 生成在线图书
quarto render

0.5 存储方式对比

格式 文件 大小 适用场景
CSV data/combined/combined_data.csv ~2.6 MB 通用、人类可读
Parquet data/combined/combined_data.parquet ~0.9 MB 大数据、列式读取、速度快
SQLite data/combined/combined_data.db ~2.3 MB SQL 查询、多表关联
Tip

三种格式已全部在 02_clean.ipynb 中生成并对比，详见 Notebook 输出。
